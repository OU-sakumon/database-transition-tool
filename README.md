# 模試データ移行ツール

模試データを効率的に入力し、JSON形式で出力するためのWebアプリケーションです。

## 機能

- 模試データの入力（科目、種類、年度、大問数）
- 複数行のデータ入力に対応
- 入力データのバリデーション
- JSON形式でのデータ出力（クリップボードにコピー）

## 使い方

1. `index.html` をブラウザで開きます
2. 各フィールドに情報を入力します：
   - **科目**: ドロップダウンから選択
   - **模試の種類**: テキスト入力
   - **年度**: 数値入力（0以上）
   - **大問数**: 数値入力（0以上、必須）
3. 「+ 行を追加」ボタンで複数のデータを入力できます
4. 「JSONをコピー」ボタンをクリックすると、入力したデータがJSON形式でクリップボードにコピーされます

## ショートカットキー

- **Ctrl+Enter**（Mac: **⌘+Enter**）: 新しい行を追加
- **Ctrl+Backspace**（Mac: **⌘+Backspace**）: 現在フォーカスがある行を削除

## 出力形式

出力されるJSONの形式は以下の通りです：

```json
[
  {
    "subject": "理系数学",
    "kind": "全統模試",
    "year": 2024,
    "count": 5
  },
  {
    "subject": "英語",
    "kind": "駿台模試",
    "year": 2024,
    "count": 4
  }
]
```
